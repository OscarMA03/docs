## Quick Start

<Steps>
  <Step title="Get API Access">
    Contact support to obtain your API key for Builder Prime integration.
  </Step>
  <Step title="Test the API">
    Use the health endpoint to verify connectivity: `GET /health`
  </Step>
  <Step title="Create Your First Lead">
    Use the create-lead endpoint to add new leads to your system.
  </Step>
  <Step title="Schedule Appointments">
    Retrieve available time slots and book appointments for your leads.
  </Step>
</Steps>

## Authentication

Most endpoints require the `BUILDER_PRIME_API_KEY` environment variable to be set. The API key is automatically used for authentication with the Builder Prime backend.
