## Update Lead

### Use Cases

- **Lead Updates**: Update existing lead information
- **Status Changes**: Change lead status or source
- **Contact Updates**: Update contact information for existing leads
- **Partial Updates**: Update only specific fields without affecting others

### Implementation Notes

- The endpoint returns the updated lead record directly from Builder Prime API
- All fields in the request body are optional - only include fields you want to update
- The `opportunityId` is required in the URL path
- Requires both `x-api-key` and `x-builder-prime-api-key` headers for authentication
