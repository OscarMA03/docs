## Create Phone Number

### Use Cases

- **Phone Number Creation**: Create a phone number with Retell inbound and outbound agents
- **Agent Assignment**: Assign specific agent versions to phone numbers
- **Webhook Setup**: Configure webhook URLs for inbound call handling
- **Provider Management**: Set up phone numbers with different providers (e.g., Twilio)

### Implementation Notes

- The endpoint creates a phone number with both inbound and outbound Retell agents
- Agent versions allow you to use specific versions of your agents
- The phone number must be in E.164 format (e.g., "+14157774444")
- Area code is required for number provisioning
- Webhook URL is optional but recommended for handling inbound calls
- Number provider determines which telephony service is used
