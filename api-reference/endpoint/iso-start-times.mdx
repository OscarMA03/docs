## Get ISO Start Times

Get only ISO start time values for appointments.

### Headers

<ParamField path="header" type="string" name="x-builder-prime-api-key" required>
  Builder Prime API key for authentication
</ParamField>

<ParamField path="header" type="string" name="x-api-key" required>
  API key for authentication
</ParamField>

<ParamField path="header" type="string" name="x-company-url-append" required>
  Company URL append value (e.g., "stage", "prod")
</ParamField>

### Request Parameters

<ParamField path="query" type="string" name="opportunityId" required>
  The opportunity ID to get start times for
</ParamField>

### Example Request

```bash
GET /iso-start-times?opportunityId=12345
x-builder-prime-api-key: <your-builder-prime-api-key>
x-api-key: <your-api-key>
x-company-url-append: <your-company-url-append>
```

### Example Response

```json
{
  "success": true,
  "opportunityId": "12345",
  "totalSlots": 3,
  "isoStartTimes": [
    "2024-01-15T09:00:00.000Z",
    "2024-01-15T10:00:00.000Z",
    "2024-01-15T11:00:00.000Z"
  ],
  "message": "Successfully extracted 3 isoStartTime values (sorted by earliest date first)"
}
```

### Error Response

```json
{
  "success": false,
  "error": "Failed to get ISO start times",
  "details": "Error message from the API"
}
```

### Response Fields

<ParamField path="response" type="boolean" name="success">
  Indicates if the request was successful
</ParamField>

<ParamField path="response" type="string" name="opportunityId">
  The opportunity ID that was queried
</ParamField>

<ParamField path="response" type="number" name="totalSlots">
  Total number of available time slots
</ParamField>

<ParamField path="response" type="array" name="isoStartTimes">
  Array of ISO start time strings, sorted by earliest date first
</ParamField>

<ParamField path="response" type="string" name="message">
  Success message describing the result
</ParamField>

### Use Cases

- **Time Selection**: Get available start times in ISO format
- **API Integration**: Use ISO times for external system integration
- **Time Validation**: Validate appointment times before scheduling
- **Sorted Results**: Get times sorted chronologically (earliest first)
