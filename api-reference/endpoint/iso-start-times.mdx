## Get ISO Start Times

Get only ISO start time values for appointments.

### Headers

<ParamField path="header" type="string" name="x-builder-prime-api-key" required>
  Builder Prime API key for authentication
</ParamField>

<ParamField path="header" type="string" name="x-api-key" required>
  API key for authentication
</ParamField>

### Request Parameters

<ParamField path="query" type="string" name="opportunityId" required>
  The opportunity ID to get start times for
</ParamField>

### Example Request

```bash
GET /iso-start-times?opportunityId=12345
x-builder-prime-api-key: <your-builder-prime-api-key>
x-api-key: <your-api-key>
```

### Example Response

```json
{
  "success": true,
  "opportunityId": "12345",
  "isoStartTimes": [
    "2024-01-15T09:00:00.000Z",
    "2024-01-15T10:00:00.000Z",
    "2024-01-15T11:00:00.000Z"
  ],
  "count": 3
}
```

### Error Response

```json
{
  "success": false,
  "error": "Failed to get ISO start times",
  "details": "opportunityId parameter is required"
}
```

### Use Cases

- **Time Selection**: Get available start times in ISO format
- **API Integration**: Use ISO times for external system integration
- **Time Validation**: Validate appointment times before scheduling
