## Gate 1 Validate All

Validate all required fields for lead creation. This endpoint checks if all necessary fields are present and valid before creating a new opportunity.

<ParamField path="query" type="string" name="firstName">
  First name to validate
</ParamField>

<ParamField path="query" type="string" name="lastName">
  Last name to validate
</ParamField>

<ParamField path="query" type="string" name="phone">
  Phone number to validate
</ParamField>

<ParamField path="query" type="string" name="email">
  Email address to validate
</ParamField>

<ParamField path="query" type="string" name="address">
  Address to validate
</ParamField>

### Example Request

```bash
GET /gate1/validate-all?firstName=John&lastName=Doe&phone=+1234567890&email=john@example.com&address=123 Main St
```

- **Pre-Creation Validation**: Validate all fields before creating a lead
