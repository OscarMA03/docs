## Create Simple Lead

Create a simplified lead with core fields only. The `secretKey` is automatically set from the `BUILDER_PRIME_API_KEY` environment variable.

<ParamField path="body" type="object" name="leadData" required>
  Simplified lead data for creation
</ParamField>

### Request Body

<ParamField path="body" type="string" name="firstName" required>
  Lead's first name
</ParamField>

<ParamField path="body" type="string" name="lastName">
  Lead's last name (optional, defaults to empty string)
</ParamField>

<ParamField path="body" type="string" name="email" required>
  Lead's email address
</ParamField>

<ParamField path="body" type="string" name="phone" required>
  Lead's phone number
</ParamField>

<ParamField path="body" type="string" name="addressLine1" required>
  Lead's address (whatever address the user provides)
</ParamField>

<ParamField path="body" type="string" name="city">
  Lead's city (optional)
</ParamField>

<ParamField path="body" type="string" name="state">
  Lead's state (optional)
</ParamField>

<ParamField path="body" type="string" name="zip">
  Lead's ZIP code (optional)
</ParamField>

### Example Request

```bash
POST /create-lead
Content-Type: application/json

{

```
