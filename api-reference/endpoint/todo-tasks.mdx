## Create To-Do Task

Create a new to-do task. The employee ID is automatically set from the `HARDCODED_EMPLOYEE_ID` environment variable.

### Headers

<ParamField path="header" type="string" name="x-builder-prime-api-key" required>
  Builder Prime API key for authentication
</ParamField>

<ParamField path="header" type="string" name="x-api-key" required>
  API key for authentication
</ParamField>

<ParamField path="header" type="string" name="x-company-url-append" required>
  Company URL append value (e.g., "stage", "prod")
</ParamField>

<ParamField path="header" type="string" name="Content-Type" required>
  application/json
</ParamField>

### Request Parameters

<ParamField path="query" type="string" name="opportunityId">
  Optional opportunity ID to associate with the task
</ParamField>

<ParamField path="body" type="object" name="taskData" required>
  Task data for creation
</ParamField>

### Request Body

<ParamField path="body" type="string" name="title" required>
  Task title
</ParamField>

<ParamField path="body" type="string" name="type" required>
  Task type (must be one of: TODO, CALL, EMAIL)
</ParamField>

<ParamField path="body" type="string" name="description">
  Task description
</ParamField>

<ParamField path="body" type="string" name="dueDate">
  Task due date (ISO format)
</ParamField>

<ParamField path="body" type="object" name="employee">
  Employee object (automatically set from environment)
</ParamField>

<ParamField path="body" type="object" name="opportunity">
  Opportunity object (set if opportunityId is provided)
</ParamField>

### Example Request

```bash
curl --request POST \
  --url 'https://proxy-server-builder-prime.onrender.com/to-do-tasks/v1?opportunityId=12345' \
  --header 'Content-Type: application/json' \
  --header 'x-builder-prime-api-key: <your-builder-prime-api-key>' \
  --header 'x-api-key: <your-api-key>' \
  --header 'x-company-url-append: <your-company-url-append>' \
  --data '{
  "title": "Follow up with lead",
  "type": "CALL",
  "description": "Call the lead to discuss their requirements",
  "dueDate": "2024-01-20T10:00:00Z"
}'
```

Or using HTTP:

```bash
POST /to-do-tasks/v1?opportunityId=12345
Content-Type: application/json
x-builder-prime-api-key: <your-builder-prime-api-key>
x-api-key: <your-api-key>
x-company-url-append: <your-company-url-append>

{
  "title": "Follow up with lead",
  "type": "CALL",
  "description": "Call the lead to discuss their requirements",
  "dueDate": "2024-01-20T10:00:00Z"
}
```

### Use Cases

- **Task Management**: Create tasks for lead follow-up
- **Workflow Automation**: Automatically create tasks based on lead status
- **Team Coordination**: Assign tasks to team members
