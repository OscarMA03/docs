## Gate 1 First Check

First validation check to determine if a new opportunity ID is needed. If all validations pass, indicates no new opportunity ID is needed.

<ParamField path="query" type="string" name="firstName">
  First name to validate
</ParamField>

<ParamField path="query" type="string" name="lastName">
  Last name to validate (optional)
</ParamField>

<ParamField path="query" type="string" name="phone">
  Phone number to validate
</ParamField>

<ParamField path="query" type="string" name="email">
  Email address to validate
</ParamField>

<ParamField path="query" type="string" name="address">
  Address to validate
</ParamField>

### Example Request

```bash
GET /gate1/first-check?firstName=John&lastName=Doe&phone=+1234567890&email=john@example.com&address=123 Main St
```
