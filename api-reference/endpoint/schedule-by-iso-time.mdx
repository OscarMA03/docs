## Schedule by ISO Time

Schedule an appointment using ISO start time.

### Headers

<ParamField path="header" type="string" name="x-builder-prime-api-key" required>
  Builder Prime API key for authentication
</ParamField>

<ParamField path="header" type="string" name="x-api-key" required>
  API key for authentication
</ParamField>

<ParamField path="header" type="string" name="Content-Type" required>
  application/json
</ParamField>

### Request Parameters

<ParamField path="query" type="string" name="opportunityId" required>
  The opportunity ID to schedule
</ParamField>

<ParamField path="body" type="object" name="timeData" required>
  ISO time data for scheduling
</ParamField>

### Request Body

<ParamField path="body" type="string" name="isoStartTime" required>
  ISO format start time for the appointment
</ParamField>

### Example Request

```bash
POST /schedule-by-iso-time?opportunityId=12345
Content-Type: application/json
x-builder-prime-api-key: <your-builder-prime-api-key>
x-api-key: <your-api-key>

{
  "isoStartTime": "2024-01-15T09:00:00.000Z"
}
```

### Example Response

```json
{
  "success": true,
  "opportunityId": "12345",
  "requestedIsoStartTime": "2024-01-15T09:00:00.000Z",
  "slotKey": "morning",

    "isoStartTime": "2024-01-15T09:00:00.000Z",
    "employeeHomeAddress": "",
    "prevAppointmentAddress": "",
    "nextAppointmentAddress": "",
    "totalTravelTimeInSeconds": null
  },
  "scheduledAppointment": {
    "id": "appt_123",
    "startTime": "2024-01-15T09:00:00.000Z",
    "status": "scheduled"
  },
  "scheduleStatus": "success",
  "message": "Successfully scheduled appointment for 2024-01-15T09:00:00.000Z"
}
```

### Error Response

```json
{
  "success": false,
  "error": "No appointment found for isoStartTime: 2024-01-15T09:00:00.000Z",
  "opportunityId": "12345",
  "requestedIsoStartTime": "2024-01-15T09:00:00.000Z"
}
```

### Use Cases

- **Precise Scheduling**: Schedule appointments with exact ISO timestamps
- **API Integration**: Use ISO times from external systems
- **Time Accuracy**: Ensure precise time scheduling across timezones
- **Slot Matching**: Automatically find and match available time slots
